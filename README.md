# Introdução:
Essa biblioteca funciona de forma parcial com a GameMode, separando sistemas, permitindo novos programadores adentrar na programação do servidor através da mesma.
Usando um gerenciador de pacotes chamado ***sampctl*** nós puxamos os dados da branch dessa biblioteca e automaticamente é adicionado na GameMode.
Os programadores que fizer alteração no código, pode aguardar a aprovação de Theus_Crazzy, para mesclar, e aguarde a GameMode ser atualizada in-game.
Abaixo estarei citando as disponibilidades que temos nesse pacote.

# Bibliotecas disponíveis:

* `commands/cmd_help` - Usado para alterar o comando /ajuda, e algumas outras coisas.
* `inventory/sacolas` - Usado para a criação de novas sacolas.
